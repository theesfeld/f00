## Usage notes

- Prefer SVG; rasterize only when a platform requires PNG.
- Do not recolor the active row away from accent green without brand review.
- Minimum clear space: ~1/8 of mark height on all sides.
- On dark UI, use `logo.svg`; on light print decks, use `logo-light.svg`.
