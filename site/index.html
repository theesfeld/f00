Why f00

One binary that covers the coreutils job and the modern listing tools — implemented entirely in Rust.

Drop-in for coreutils ls

Full GNU flag surface. Use f00 --gnu (or alias ls='f00 --gnu') when you need script-shaped, coreutils-compatible output. Pipes and non-TTY stdout switch to that safe mode automatically.

Meets and exceeds eza and lsd

Icons, git status, modern long listings, and tree view on a TTY — plus better wall times than eza and lsd on the common short and long workloads we measure. You keep the extras without giving up coreutils coverage.

Written in Rust

Memory-safe systems code: parallel metadata, Linux statx and optional io_uring, a clear crate layout, and a single static-friendly binary. Rust is not a footnote — it is how f00 is built.

Tree and JSON that are real surfaces

--tree for hierarchy at a glance. --json / -j for tooling: rich fields, pretty and colored on a TTY, compact plain JSON for jq when color is off.

What you get

GNU surface + auto safety

Full flag set. Non-TTY behaves like --gnu. Escape hatch: --no-gnu.

JSON

First-class --json / -j with size, mode, inode, owner, git status, and more.

Tree

--tree for directory structure without a second tool.

Icons · git · colors

Nerd Font icons, git column, long-format colors — on by default when stdout is a TTY.

f00-tui

Optional dual-pane browser binary in the same release tarball.

Self-update

f00 --update and --check-update from GitHub Releases with checksums.

Install

One command, or your usual package manager.

$ curl -fsSL https://f00.sh/install.sh | bash

Recommended · ~/.local/bin · f00 + f00-tui · never touches /bin/ls 

# pin a release
curl -fsSL https://f00.sh/install.sh | F00_VERSION=__VERSION__ bash

# custom directory
curl -fsSL https://f00.sh/install.sh | INSTALL_DIR=$HOME/bin bash

# optional: also put ls → f00 on PATH (not /bin/ls)
curl -fsSL https://f00.sh/install.sh | F00_INSTALL_LS=1 bash
# later
f00 --update
f00 --check-update
$ nix profile install github:theesfeld/f00

Builds from source via the flake

nix run github:theesfeld/f00 -- -la
$ cargo install f00 --locked
